# Installation Guide

## Requirements

To run this project, you only need:

* A modern web browser (Google Chrome, Firefox, Brave, Opera, etc)

## Installation

1. Download or clone this repository.

```bash
git clone https://github.com/yourusername/bakery-management-system.git
```

2. Navigate to the project folder.

3. Open the `index.html` file in your preferred web browser.

## Alternative Method

You can also download the project as a ZIP file:

1. Click on **Code**.
2. Select **Download ZIP**.
3. Extract the files.
4. Open `index.html` in your browser.

## Notes

This project was developed as a front-end application using HTML, CSS, and JavaScript. No additional software, database, or server configuration is required.
