# User Guide

## Overview
The Bakery Management System is a web application designed to assist bakery businesses in organizing inventory, products, customers, and operational expenses.

## Dashboard
The dashboard provides an overview of the bakery's operations and important management information.

## Ingredient Management
Users can:
* Register new ingredients.
* View registered ingredients.
* Manage ingredient information.

## Product Management
Users can:
* Register desserts and bakery products.
* View product information.
* Organize the product catalog.

## Inventory Management
The inventory section allows users to:
* Monitor available items.
* Track stock information.
* Improve inventory organization.

## Order Management
The Order Management module is responsible for handling customer orders, from creation to completion.
### Features
* Create new orders
* Filter orders by:
  - All Orders
  - In Production
  - Completed
* Sort orders by:
  - Most Recent
  - Oldest
  - Customer Name
  - Product Name
* Search orders by customer name or product name

### Creating a New Order
To create a new order, click New Order and complete the following steps:
**1. Order Information**
- Select the product type
- Choose the delivery date
**2. Customer Information**
- Customer name
- Phone number
- Address
- Estimated delivery distance
**3. Finalization**
- Enter the amount charged for the order
- Choose whether delivery fees will be charged to the customer
- If delivery fees should be charged, enable the **Charge Delivery Fee** option
  
### Order Status
Orders can be organized into two categories:
- In Production – Orders currently being prepared.
- Completed – Finished orders that are ready for delivery or have already been delivered.

## Financial Dashboard
The Financial Dashboard provides an overview of the business's financial performance.
### Features
- Total Income
- Total Expenses
- Total Profit
- Order History

### Financial Information
The system automatically calculates financial data based on completed orders and registered expenses.
Users can monitor:
- Total revenue generated
- Total operational expenses
- Net profit
- Detailed financial history of completed orders

> Note: Financial data is automatically generated and cannot be edited directly from this module.

## Settings
The Settings module allows users to customize the system according to their business needs.
### 1. System Identity
Customize the visual identity of the application.
**Available Options**
- System Name
- System Logo/Image

> If no image is selected, the application will use the default bakery icon.

### 2. Menu Organization
Users can customize the order of the navigation menu.
- Features
- Drag and drop menu items
- Save custom menu order
- Restore default layout

This allows each user to organize the application according to their workflow preferences.

### 3. Delivery Fee Configuration
Configure the delivery cost calculation used during order creation.
**Available Options**
- Set the price per kilometer
The default value is:
**R$ 0.70 per kilometer**
This value is automatically used when calculating delivery fees for customer orders.

## Purpose
The main objective of the system is to replace traditional paper-based management methods with a more organized and user-friendly digital solution.
