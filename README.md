# 🎯 | Challenge  <a id="Challenge"></a>
Small bakery businesses often manage inventory, ingredients, customers, and expenses using manual methods such as paper records or spreadsheets. These approaches can lead to disorganized information, operational errors, and reduced productivity.
The challenge of this project was to develop a digital solution capable of centralizing this information into a single platform, making financial and operational management more efficient and accessible.

# 💡 | Solution  <a id="Solution"></a>
The Bakery Management System was developed as a web application designed to support bakery management and daily operations.

Main features include:
- Ingredient registration and management.
- Product and dessert registration.
- Inventory control.
- Customer registration.
- Shipping and expense calculation.
- Dashboard with an overview of business operations.

The goal is to replace traditional management methods with a more organized, user-friendly, and efficient system.
